## Installation

```bash
$ pnpm install
```

## Running DB Migration & Seeder
```bash
# generate
$ pnpm drizzle-kit generate

# migrate
$ pnpm drizzle-kit migrate

# studio
$ pnpm drizzle-kit studio

# seeder
$ pnpm run db:seed
```


## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
